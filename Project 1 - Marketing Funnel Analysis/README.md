# Marketing Funnel & Customer Journey Analysis
## Project Overview 
This project analyzes a simulated ecommerce customer journey to identify customer drop-off points, evaluate traffic source performance, and provide actionable marketing recommendations through an interactive dashboard.
## Business Problem
The objective was to understand customer behavior throughout the marketing funnel, identify the largest drop-off points, evaluate acquisition channels, and recommend improvements to increase conversions.
## Objectives
- Analyze customer acquisition
- Measure purchase conversion
- Evaluate traffic source performance
- Compare mobile vs desktop users
- Build an executive dashboard
- Generate marketing recommendations
## Dataset
- 500 simulated ecommerce customers
- Generated using ChatGPT
Includes:
- Traffic Source
- Device
- Viewed Website
- Viewed Product
- Added to Cart
- Checkout Started
- Purchased
- Purchase Value
## Tools Used
- Google Sheets
- Looker Studio
- ChatGPT
- Excel
## KPI Summary
| KPI             |  Value |
| --------------- | -----: |
| Customers       |    500 |
| Purchases       |     64 |
| Revenue         | $5,288 |
| Conversion Rate |  12.8% |
## Key Insights
- Customer Acquisition
Instagram generated the largest share of website visitors, making it the strongest acquisition channel.
- Device Usage
Nearly 70% of visitors accessed the website through mobile devices, highlighting the importance of mobile optimization.
- Funnel Performance
Only 64 out of 500 visitors completed a purchase, resulting in a conversion rate of 12.8%.
- Revenue
The analyzed campaigns generated approximately $5,288 in simulated revenue.
## Recommendations
- Increase investment in Instagram campaigns because it delivers the highest visitor volume.
- Improve checkout flow to reduce purchase abandonment.
- Continue optimizing the mobile experience due to the high proportion of mobile visitors.
- Test underperforming acquisition channels using new creatives and audience targeting.
